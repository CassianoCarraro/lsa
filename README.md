Simulação de compilador que realiza a análise léxica e sintática de um arquivo fonte em C. O subconjunto da linguagem C reconhecida é:
- Comando while.
- Comando do-while.
- Comando for.
- Comando if.
- Atribuição.
- Comando switch-case.
- Expressões.
- Blocos de comandos.
- Declarações e chamadas de funções.
